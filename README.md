
# STM32F103ZET6 智能小车
**电机闭环调速 | 舵机激光雷达 | FreeRTOS | VL53L0X | MPU6050 | OLED | ESP8266**

基于 STM32F103ZET6 开发的多功能智能小车平台，集成 PID 调速、虚拟激光雷达、姿态检测、多任务调度、无线通信，支持二次开发、课程设计、ROS 联动。

---

## 🌟 项目简介
本项目基于 **STM32F103ZET6** 主控，使用 **Keil MDK5 + HAL 库 + FreeRTOS** 开发，实现一套完整的嵌入式机器人底层平台。

功能包含：
- 电机编码器 **PID 闭环调速**
- **SG90 + VL53L0X** 实现简易激光雷达扫描
- **MPU6050** 陀螺仪姿态获取
- **OLED** 实时状态显示
- **ESP8266** 无线串口扩展
- 模块化驱动 + 多任务调度

适合：机器人入门、嵌入式实战、自动避障、SLAM 扩展、课程设计。

## 🧩 硬件清单
| 模块 | 说明 |
|------|------|
| **主控** | STM32F103ZET6 |
| **电机** | 直流减速电机 + 编码器 |
| **电机驱动** | L298N / TB6612 / 其他驱动板 |
| **激光雷达** | VL53L0X（ToF 测距） + SG90 舵机 |

| **PWM 扩展** | PCA9685 |
| **显示** | 0.96寸 I2C OLED |
| **通信** | ESP8266（串口无线） |
| **电源** | 12V 电池组 |

## ⚙️ 标准接线表（I2C 共线）
| STM32 引脚 | 功能 | 外接模块 |
|-----------|------|----------|
| PB6 | I2C1_SCL | VL53L0X / MPU6050 / PCA9685 / OLED |
| PB7 | I2C1_SDA | VL53L0X / MPU6050 / PCA9685 / OLED |
| PA0 | TIM2_CH1 | SG90 舵机 PWM |
| PA8 | TIM1_CH1 | 电机1 PWM |
| PA9 | TIM1_CH2 | 电机2 PWM |
| PA10 | USART1_RX | ESP8266 TX |
| PA9 | USART1_TX | ESP8266 RX |
| PB0 / PB1 | 编码器 | 电机编码器 A/B 相 |
| 3.3V | 电源 | 所有传感器模块 |
| GND | 共地 | 所有系统共地 |

## 🚀 主要功能
✅ **电机 PID 闭环调速**（编码器反馈）
✅ **舵机扫描 + VL53L0X 简易激光雷达**（可发 /scan 数据）
✅ **MPU6050 姿态解算**（加速度、陀螺仪、角度）
✅ **FreeRTOS 多任务调度**
✅ **OLED 实时显示**（速度、距离、角度、状态）
✅ **ESP8266 无线串口通信**
✅ **运动学正逆解**
✅ 模块化驱动，易于扩展二次开发

## 🧪 开发环境
- **IDE**: Keil MDK5 (5.28+)
- **库**: STM32 HAL Library
- **OS**: FreeRTOS
- **调试器**: ST-Link
- **主控**: STM32F103ZET6 (72MHz)
